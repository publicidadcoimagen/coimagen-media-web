### Integración Jotform Agent como CTA Principal
El asistente Jotform ya está activo como widget flotante. Para hacerlo CTA principal:
```javascript
// Trigger programático para abrir el Jotform Agent
window.dispatchEvent(new CustomEvent('openJotformAgent'));
// o: el widget expone un método global para abrirlo
```
Se debe investigar el API de Jotform embed para trigger manual desde botones.
